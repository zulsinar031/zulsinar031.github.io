Project 01

Multimodal RAG Pipeline

Retrieval-augmented generation system that understands and answers questions across text documents, PDFs, and images using a unified pipeline. Uses CLIP embeddings to map text and images into the same vector space — enabling a single search index to handle all content types seamlessly. Automatically routes queries to a vision model when images are retrieved, and a text model otherwise. Built with a React frontend, FastAPI backend, and Docker for containerised deployment.

CLIP RAG Groq Multimodal LLM Docker FastAPI
Project 02

Airbnb Market Segmentation & Price Prediction

Large-scale analysis of 49,000+ Australian Airbnb listings using PySpark on a YARN cluster with data stored on Google Cloud Storage. Covers four ML problem types: room type classification (Random Forest, 84.65% accuracy), price regression (Gradient Boosting, RMSE $88.91), market segmentation via K-Means into 5 distinct tiers, and association rule mining with FP-Growth to uncover pricing patterns across neighbourhoods.

PySpark GCP / YARN Classification Regression Clustering FP-Growth
Project 03

Australian Unemployment Forecasting & Economic Analysis

End-to-end macroeconomic time series analysis using real ABS and RBA datasets covering unemployment, CPI, and the RBA cash rate from 2011–2024. Implemented STR decomposition from scratch (spline trend + seasonal averaging), validated against statsmodels STL. ARIMA(2,2,1) selected via AIC grid search outperformed Gradient Boosting by 29.3% RMSE. Built SARIMAX scenario models simulating RBA rate cut vs hold policies to forecast unemployment through mid-2025.

ARIMA SARIMAX STR Decomposition Gradient Boosting ABS / RBA Data Scenario Modelling

Project 04

Job Application Agent

LangGraph agentic pipeline that reads a job posting URL, extracts structured role requirements, researches the company via live web search, and generates a tailored cover letter in under 30 seconds. Built with a modular node architecture — scraper, analyzer, researcher, and writer — each powered by Groq's llama-3.3-70b for fast inference. Containerised with Docker for portable deployment.

LangGraph Groq Tavily Agentic AI Docker Python

Competitor Analysis Agent

LangGraph multi-node agent that takes a company name, identifies 3–5 competitors using an LLM planner, runs live Tavily web searches per competitor in a loop, and synthesizes findings into a structured Markdown report — covering feature comparisons, strengths and weaknesses, and pricing breakdowns. Useful for product teams, startups, and consultants who need rapid competitive intelligence.
